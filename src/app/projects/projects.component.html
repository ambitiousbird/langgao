Smart Wearable Systems

My past research involves security and medical applications of embedded sensor devices.

Multiple device authentication with accelerometer data

As the volume of many devices becomes smaller and smaller, typing password to authentication device communication is no longer doable in many cases. While the plausibility of pairing up two devices using accelerometer data has been explored, little is known about the authentication of multiple devices with acceleration sensor. In this research we studied various possible algorithms to solve this problem.

Motion analysis of Mild Cognitive Impairment (MCI) patients using accelerometer

MCI is an intermediate stage of brain malfunction between normal aging and more serious dementia, of which the most common one is Alzheimer’s disease. In our study of MCI patients’ hand movement patterns, we found more pauses and direction changes compared to normal people. We believe these findings open up new opportunities for future research on MCI diagnosis.

Machine Learning and Big Data
Emotion analysis on text mining for election prediction

In this project, I measured the public opinions on presidential candidates based on eight emotions: anger, anticipation, disgust, fear, joy, sadness, surprise, trust, by applying text mining on the most recent tweets at the time. Compared to traditional text mining methods which categorize emotions simply as neutral, negative, and positive, this method provides more details and perspectives for emotion analysis.


  • Collected and analyzed 1.25M tweets using Python and Twitter API
  • Performed sentiment analysis using the open source dictionary SentiWordNet

Towards Learning based Reasoning Systems

In this project, we investigated several learning based reasoning methods. First, we explored the ways to build explicit rules for problem solving. It corresponds to the learning process of conscious reasoning and the decision tree based technique is also applied the project. As conscious reasoning may not solve certain problems in our practice, we also investigated making implicit rules for problem solving. More specifically, we utilized the deep neural network to simulate the process of unconscious reasoning. Lastly, we explored research progress on emotion and creativity.


  • Used decicion tree on classification of Iris flower using the Iris Plants database
  • Applied Deep Neural Network on the MNIST dataset, CIFAR dataset and PASCAL VOC 2007 dataset.
